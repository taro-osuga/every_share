# every_share（仮）

## 概要

旅先などで持って行くのはめんどくさい。レンタル出来たらなあと思う人と、
現地在住で普段あまり使っていない物を使ってもらいたい人をマッチングさせるアプリ。

## コンセプト

個人間商品レンタルアプリ

## バージョン

Ruby 2.6.5
Rails 5.2.4
